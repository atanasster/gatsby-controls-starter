# start project

1. clone

```
  git clone git@github.com:atanasster/gatsby-controls-starter.git
```

2. move to project folder :)

```
cd gatsby-controls-starter
```

3. install dependencies
```
  yarn
```

4. start
```
  yarn start
```

5. build
```
  yarn build
```
